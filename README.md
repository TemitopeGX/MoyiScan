### Prerequisites

### Installation

1. Clone the repository

```bash
git clone https://github.com/username/moyiscan.git
```

2. Install dependencies

```bash
cd moyiscan
npm install
```

3. Set up environment variables


4. Start the development server

